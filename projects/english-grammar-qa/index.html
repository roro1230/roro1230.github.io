English Grammar Q&A Assistant

A Retrieval-Augmented Generation (RAG) based system for English grammar question answering.

← Back to Portfolio GitHub

Overview

This project implements a grammar question-answering assistant using LangChain, FAISS, and OpenAI models. The system retrieves relevant grammar rules from documents before generating grounded explanations.

Architecture

Features

  • PDF-based knowledge ingestion and text chunking
  • Semantic retrieval with FAISS
  • RAG pipeline with OpenAI LLMs
  • Interactive Streamlit-based UI

Demo

Tech Stack

Python, LangChain, FAISS, OpenAI API, HuggingFace Embeddings, Streamlit
